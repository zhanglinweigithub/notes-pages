                                                                                                                                        tar、gz压缩

                                                                                                                                        .tar：归档文件，既简单的将文件组装到一个.tar的文件内，并没有太多文件体积的减少，仅仅是简单的封装

                                                                                                                                        .gz：gzip格式压缩文件，即使用gzip压缩算法将文件压缩到一个文件内，可以极大的减少压缩后的体积

                                                                                                                                        tar [-c -v -x -f -z -C] 要压缩的文件1，要压缩的文件2……要压缩的文件N

tar -cvf test.tar 1.txt 2.txt 3.txt
tar -zcvf test.tar.gz 1.txt 2.txt 3.txt

                                                                                                                                        -z如果使用，需处于选项位第一个。-f必须在选项位最后一个

                                                                                                                                        提示

                                                                                                                                        -c： 创建压缩文件，用于压缩模式

                                                                                                                                        -v： 显示压缩、解压过程，用于查看进度

                                                                                                                                        -x： 解压模式

                                                                                                                                        -f： 要创建的文件，或要解压的文件，-f必须在所有选项中位置处于最后一个

                                                                                                                                        -z： gzip模式，不使用-z就是普通的tarball格式

                                                                                                                                        -C： 选择解压的目的地，用于解压模式

                                                                                                                                        zip压缩

                                                                                                                                        zip [-r] 参数1 参数2 ……参数N

zip test.zip a.txt b.txt c.txt
zip -r test.zip test zhang a.txt
                                                                                                                                        • -r： 被压缩的包含文件夹时，需要用-r选项（和rm命令的-r效果一致）

                                                                                                                                        tag、gz解压缩

                                                                                                                                        tar [-c -v -x -f -z -C] 要解压的文件

tar -xvf test.tar
tar -xvf test.tar -C .home.zhang
tar -zxvf test.tar.gz -C .home.zhang

                                                                                                                                        -f必须在最后一位，-z必须在开头，-C必须单独使用和解压所需其他参数分开

                                                                                                                                        提示

                                                                                                                                        -c： 创建压缩文件，用于压缩模式

                                                                                                                                        -v： 显示压缩、解压过程，用于查看进度

                                                                                                                                        -x： 解压模式

                                                                                                                                        -f： 要创建的文件，或要解压的文件，-f必须在所有选项中位置处于最后一个

                                                                                                                                        -z： gzip模式，不使用-z就是普通的tarball格式

                                                                                                                                        -C： 选择解压的目的地，用于解压模式

                                                                                                                                        zip解压缩

                                                                                                                                        unzip [-d] 参数

unzip test.zip
unzip test.zip -d .home.zhang

                                                                                                                                        -d： 指定解压去的位置（同tar的-C）

                                                                                                                                        ls 列出文件夹信息

                                                                                                                                        ls [-l -h -a] [参数]

ls
ls /data

                                                                                                                                        提示

                                                                                                                                        • 参数：被查看的文件夹，不提供参数，表示查看当前工作目录
                                                                                                                                        • -l，以列表形式查看
                                                                                                                                        • -h，配合-l，以更加人性化的方式显示文件大小
                                                                                                                                        • -a，显示隐藏文件
                                                                                                                                        • -t，按修改时间排序文件和目录
                                                                                                                                        • -r，以相反的顺序列出文件和目录
                                                                                                                                        • -R，递归列出子目录的内容
                                                                                                                                        • -S，按文件大小排序
                                                                                                                                        • --color，启用彩色输出，以区分不同类型的文件
                                                                                                                                        • -i，显示文件的inode号
                                                                                                                                        • -d，仅显示目录本身，而不显示其内容

                                                                                                                                        cp 功能：复制文件、文件夹

                                                                                                                                        cp [-r] 参数1 参数2

cp a.txt b.txt		# 复制当前目录下a.txt为b.txt
cp a.txt test/		# 复制当前目录a.txt到test文件夹内 
cp -r test test2	# 复制文件夹test到当前文件夹内的test2文件夹内

                                                                                                                                        提示

                                                                                                                                        • 参数1，被复制的
                                                                                                                                        • 参数2，要复制去的地方
                                                                                                                                        • 选项：-r，可选，复制文件夹使用

                                                                                                                                        提示

                                                                                                                                        • 参数1：被移动的
                                                                                                                                        • 参数2：要移动去的地方，参数2如果不存在，则会进行改名

                                                                                                                                        安装

                                                                                                                                        使用

                                                                                                                                        # sz 下载
sz [选项] 文件
sz zlw.jar

# rz 上传
rz [选项]
rz -be
                                                                                                                                        • sz选项：
                                                                                                                                          • -a：以文本方式传输
                                                                                                                                          • -b：以二进制方式传输
                                                                                                                                          • -e：对控制字符转义，这可以保证文件传输正确。
                                                                                                                                        • rz选项：
                                                                                                                                          • -b：以二进制方式，默认为文本方式
                                                                                                                                          • -e：对所有控制字符转义

                                                                                                                                        提示

                                                                                                                                        • rz 命令通常会加上 -be 选项
                                                                                                                                        • 若上传时控制台输出乱码，关闭当前链接窗口，重新打开新窗口进行上传
                                                                                                                                        • xshell直接将文件拖到登录窗口即可调用rz进行文件传输

                                                                                                                                        rm 删除文件、文件夹

                                                                                                                                        rm [-r -f] 参数 . . . 参数

                                                                                                                                        提示

                                                                                                                                        • 参数：支持多个，每一个表示被删除的，空格进行分隔
                                                                                                                                        • 选项：
                                                                                                                                          • -r，删除文件夹使用
                                                                                                                                          • -f，强制删除，不会给出确认提示，一般root用户会用到

                                                                                                                                        which 查看命令的程序本体文件路径

                                                                                                                                        which 参数

                                                                                                                                        提示

                                                                                                                                        • 参数：被查看的命令

                                                                                                                                        find 搜索文件

                                                                                                                                        find 路径 -name 参数

find / -name zhang
find /data -name zhang
find /data -name *zhang

                                                                                                                                        提示

                                                                                                                                        • 路径，搜索的起始路径
                                                                                                                                        • 参数，搜索的关键字，支持通配符 *， 比如：*test表示搜索任意以test结尾的文件

                                                                                                                                        grep 过滤关键字

                                                                                                                                        grep [-n -i -o] 关键字 文件路径

grep "L" ./test.txt
grep "^i" ./test.txt -i -n -o

                                                                                                                                        提示

                                                                                                                                        • 选项：

                                                                                                                                          • -n，可选，表示在结果中显示匹配的行的行号
                                                                                                                                          • -i，可选，不区分大小写
                                                                                                                                          • -o，只显示被匹配到的关键字, 而不是将整行的内容都输出

                                                                                                                                        • 关键字，必填，表示过滤的关键字，带有空格或其它特殊符号，建议使 用""将关键字包围起来

                                                                                                                                        • 文件路径，必填，表示要过滤内容的文件路径，可作为内容输入端口

                                                                                                                                        文件路径，可以作为管道符的输入

                                                                                                                                        wc 统计

                                                                                                                                        wc [-c -m -l -w] 文件路径

wc -m ./zhang.txt
wc -w ./zhang.txt

                                                                                                                                        提示

                                                                                                                                        • 选项：
                                                                                                                                          • -c，统计bytes数量 选项
                                                                                                                                          • -m，统计字符数量 选项
                                                                                                                                          • -l，统计行数 选项
                                                                                                                                          • -w，统计单词数量
                                                                                                                                        • 文件路径，被统计的文件，可作为内容输入端口

                                                                                                                                        文件路径，可以作为管道符的输入

                                                                                                                                        管道符 | 将符号左边的结果，作为符号右边的输入

                                                                                                                                        ps -ef | grep java
cat a.txt | grep zhang	# 将cat a.txt的结果，作为grep命令的输入， 用来过滤zhang关键字
cat a.txt | grep zhang | grep linwei	# 支持嵌套

                                                                                                                                        >表示覆盖输出

                                                                                                                                        su 切换用户

                                                                                                                                        su [-] [用户]

su -
su - zhang

                                                                                                                                        提示

                                                                                                                                        -：表示切换后加载环境变量，建议带上

                                                                                                                                        用户：可以省略，省略默认切换到 root 用户

                                                                                                                                        sudo 让普通命令带有 root 权限

                                                                                                                                        sudo 其它命令

sudo ls
sudo rm -rf /*

                                                                                                                                        提示

                                                                                                                                        需要以 root 用户执行 visudo 命令，增加配置方可让普通用户有 sudo 命令的执行权限

                                                                                                                                        比如：

                                                                                                                                        zhang ALL=(ALL)	NOPASSWD: ALL

                                                                                                                                        在visudo内配置如上内容，可以让zhang用户，无需密码直接使用sudo

                                                                                                                                        chmod 修改文件、文件夹权限

                                                                                                                                        chmod [-R] 权限 参数

chmod 777 ./start.sh
chmod -R 777 ./start

                                                                                                                                        提示

                                                                                                                                        选项-R，设置文件夹和其内部全部内容一样生效

                                                                                                                                        参数，被修改的文件、文件夹

                                                                                                                                        权限，要设置的权限，比如755，表示：rwxr-xr-x

                                                                                                                                        u表示user所属用户权限，g表示group组权限，o表示other其他用户权限

                                                                                                                                        chown 修改文件、文件夹所属用户、组

                                                                                                                                        chown [-R] [用户] [:] [用户组] 文件或文件夹

chown root hello.txt # 将hello.txt所属用户修改为root
chown :root hello.txt # 将hello.txt所属组修改为root
chown root:zhang hello.txt # 将hello.txt所属用户修改为root，用户组为zhang

                                                                                                                                        提示

                                                                                                                                        选项：

                                                                                                                                        -R：同chmod

                                                                                                                                        用户：修改所属用户

                                                                                                                                        用户组：修改所属用户组

                                                                                                                                        ：用于分割用户和用户组

                                                                                                                                        创建用户组

                                                                                                                                        groupadd 用户组名

groupadd zhang
groupadd lin

                                                                                                                                        需要 root 权限

                                                                                                                                        删除用户组

                                                                                                                                        groupdel 用户组名

groupdel zhang
groupdel lin

                                                                                                                                        需要 root 权限

                                                                                                                                        查看系统全部用户组

                                                                                                                                        getent group

                                                                                                                                        提示

                                                                                                                                        信息从左至右分别为： 组名称：组认证：组ID

                                                                                                                                        查看系统全部用户

                                                                                                                                        getent passwd

                                                                                                                                        ssh 远程登录到其他机器

                                                                                                                                        ssh [-i -l -p] remote_host

ssh 127.0.0.1 # 默认使用当前登录的用户名
ssh -l user 127.0.0.1 # 如果用户名为当前登录用户，则可以省略 -l
ssh root@127.0.0.1
ssh -p 222 user@127.0.0.1
ssh -i ~/.ssh/id_rsa.pub user@127.0.0.1

                                                                                                                                        提示

                                                                                                                                        -i，指定身份验证文件（公钥），用于身份验证，也可以通过密码进行身份验证

                                                                                                                                        -l，指定登录用户名，如果不指定，将使用本地登录用户名

                                                                                                                                        -p，指定远程 SSH 服务器监听的端口号，默认为 22

                                                                                                                                        SSH 密钥对登录：

                                                                                                                                        ssh-keygen -t rsa # 生成密钥对，按提示输入要保存密钥对的文件名、密码等信息
ssh-copy-id user@127.0.0.1 # 将公钥上传到远程主机的 authorized_keys 文件
ssh user@127.0.0.1 # 登录远程主机

                                                                                                                                        输入远程主机的登录密码，将会自动将密钥上传到远程主机的 ~/.ssh/authorized_keys 文件中

                                                                                                                                        scp 复制文件到远程主机

                                                                                                                                        == 需要知道远程主机的密码 ==

                                                                                                                                        # 复制文件到远程主机
scp [-c -P -r -p] /path/to/source/file user@server-ip:/path/to/destination/directory

scp logs.tar.gz root@127.0.0.2:/root/
scp -r /tmp/ root@127.0.0.2:/root/
scp -P 5151 logs.log root@127.0.0.2:/tmp

# 文件从远程系统复制到本地系统
scp [-c -P -r -p] user@server-ip:/path/to/source/file /path/to/local/directory

scp -P 5151 root@172.29.9.33:/etc/passwd /root/
scp -P 5151 -r  root@172.29.9.33:/tmp /root/

                                                                                                                                        提示

                                                                                                                                        -c，这会在复制过程中压缩文件或目录

                                                                                                                                        -P，如果默认 SSH 端口不是 22，则使用此选项指定 SSH 端口

                                                                                                                                        -p，保留文件的访问和修改时间

                                                                                                                                        -r，此选项递归复制目录及其内容

                                                                                                                                        scp 命令的更多选项，可以查看手册，命令是：man scp
